## Rendering

- [Stride.DebugShapes](https://github.com/Doprez/Stride.DebugShapes) - Runtime debug shape rendering (lines, boxes, spheres) drawn directly in the Stride scene
  - [XenkoDebugRendering (legacy)](https://github.com/profan/XenkoDebugRendering/tree/master/DebugRendering) - Original Xenko-era debug shape renderer
- [StrideFogOfWarPlus](https://github.com/herocrab/StrideFogOfWarPlus) - Fog of war system for MOBA, platformer, and RTS games without line-of-sight
- [StrideOutlineRenderer](https://github.com/SoulRider/StrideOutlineRenderer) - Post-process outline renderer highlighting selected or highlighted objects in Stride
- [XenkoByteSized](https://github.com/profan/XenkoByteSized) - Bite-sized example scenes including various procedural rendering techniques
- [Stride.ShaderExplorer](https://github.com/tebjan/Stride.ShaderExplorer/releases) - Desktop tool to browse and inspect all built-in Stride shader source implementations
- [XenkoFlowingWater](https://github.com/TomGroner/XenkoFlowingWater) - Flowing water surface shader with normal animation and refraction
- [XenkoVoxelGI](https://github.com/WhyPenguins/XenkoVoxelGI) - VXGI-inspired voxel global illumination prototype (partially merged into Stride v4 beta)
- [XenkoVoxelExample](https://github.com/jason-wilmans/XenkoVoxelExample) - Minecraft-style voxel brick world rendered in Stride
- [Xenko.CustomRootRenderFeature](https://github.com/tebjan/Xenko.CustomRootRenderFeature) - Shows how to plug custom shaders correctly into Stride's render pipeline
- [StrideTriplanarShader](https://github.com/herocrab/StrideTriplanarShader) - Triplanar texture projection shader removing visible UV seams on complex geometry
- [StrideMeshOutlineRenderFeature](https://github.com/herocrab/StrideMeshOutlineRenderFeature) - Mesh outline render feature using a two-pass stencil technique
- [Object Info Render Target](https://github.com/Basewq/XenkoProofOfConcepts/tree/master/ObjectInfoRenderTargetExample) - Renders per-object data (ID, material) to a dedicated render target for selection/effects
- [Screen Space Decals](https://github.com/Basewq/XenkoProofOfConcepts/tree/master/ScreenSpaceDecalExample) - Projects decal textures onto geometry via screen-space depth reconstruction
- [Screen Space Decals with Root Renderer](https://github.com/Basewq/XenkoProofOfConcepts/tree/master/ScreenSpaceDecalRootRendererExample) - Screen-space decals using a custom `RootRenderFeature` for better pipeline integration
