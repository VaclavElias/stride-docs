# Games and demos

## Released Games

- [Distant Worlds 2](https://store.steampowered.com/app/1531540/Distant_Worlds_2/) - Vast, pausable real-time 4X space strategy game
- [FPS Infinite](https://store.steampowered.com/app/1256380/FPS_Infinite/) - Procedural FPS with VR support
- [Snaaker Friends](https://store.steampowered.com/app/1443760/Snaaker__Friends/) - Multiplayer Snake-inspired game
- [Children of the Galaxy](https://www.emptykeys.com/games/children-of-the-galaxy/) - Turn-based strategy game
- [Virtual Desktop](https://www.vrdesktop.net/) - VR desktop streaming app powered by Stride

## Open-Source Games and Prototypes

- [Starbreach](https://github.com/stride3d/Starbreach) - First/Third Person Shooter (official Stride demo)
- [Rollerghoster](https://github.com/Aggror/RollerGhosterOpen) - Race against online and local ghosts on procedural tracks by Aggror Jorn
- [Rise of the Undeaf](https://github.com/manio143/RiseOfTheUndeaf) - Puzzle platformer
- [Basic Platformer](https://github.com/Doprez/stride-platformer) - Simple 3D platformer WIP
- [Zerobot](https://github.com/joaovsq/Zerobot) - Robot action game prototype
- [BallsOfSteel](https://github.com/Kryptos-FR/BallsOfSteel) - Pinball-style game
- [Asteroids (Stride)](https://github.com/LanceJZ/Stride-Asteroids) - Asteroids clone in Stride
- [Asteroids (community)](https://github.com/mmujic2/StrideAsteroids) - Another Asteroids clone
- [Asteroids (Xenko)](https://github.com/LanceJZ/Xenko_Asteroids) - Original Xenko-era Asteroids
- [XenkoSpaceShooter](https://github.com/jayrulez/XenkoSpaceShooter) - Space shooter prototype
- [TurnBasedBattleSim](https://github.com/jayrulez/TurnBasedBattleSim) - Turn-based battle simulator prototype
- [Harrowing Flight](https://forums.stride3d.net/t/harrowing-flight-student-game-project/1823) - Student game project
- [Flappy Bird](https://nedreid.itch.io/flappy-bird-stride) - Flappy Bird clone with source code
- [VVVVarkle](https://github.com/BenTec3d/VVVVarkle/releases/tag/v1.0) - Dice game rendered with Stride via vvvv
- [vvvv Showcase](https://visualprogramming.net/#Showcase) - Large-scale interactive media projects made with vvvv
