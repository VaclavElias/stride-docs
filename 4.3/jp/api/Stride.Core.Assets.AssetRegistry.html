Remarks

The difference between this one and CanPropertyHandleContent(Type, out IReadOnlyList<Type>) is that this one returns the asset types, meaning the types that would be compiled into 'content' to be used at runtime. This means that the types contained in Stride.Core.Assets.AssetRegistry.AssetTypes are not directly assignable to propertyType
