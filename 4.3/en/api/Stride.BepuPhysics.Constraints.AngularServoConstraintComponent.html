ServoMaximumForce

The maximum force that the constraint can apply to move towards the target.

public float ServoMaximumForce { get; set; }

Property Value

float

Remarks

This value is specified in terms of force: a change in momentum over time. It is approximated as a maximum impulse (an instantaneous change in momentum) on a per-substep basis. In other words, for a given velocity iteration, the constraint's impulse can be no larger than ServoMaximumForce * dt where dt is the substep duration.
